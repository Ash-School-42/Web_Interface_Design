# Personal Portfolio Website

## Project Overview
This is a personal portfolio website created using HTML and CSS. The website showcases my projects, skills, and contact information. It is designed to be simple, responsive, and easy to navigate.

## Features
- Homepage with introduction and navigation menu
- Projects section highlighting selected work
- Contact section for communication
- Styled using external CSS
- Hosted using GitHub Pages

## Technologies Used
- HTML5
- CSS3
- GitHub Pages (for hosting)
- XAMPP (for local testing)
