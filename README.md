# Personal Portfolio Website

## Project Overview
This is a personal portfolio website created using HTML and CSS. The website showcases my projects, skills, and contact information. It is designed to be simple, responsive, and easy to navigate, providing a professional online presence.

## Features
- **Homepage**: Introduction and navigation menu for easy access.
- **Projects Section**: Highlights selected work with links to detailed project descriptions.
- **Contact Section**: A simple contact form for communication.
- **Responsive Design**: Optimized for both mobile and desktop viewing.
- **Styled Using External CSS**: Ensures a clean separation of content and design.
- **Hosted on GitHub Pages**: Easy access for anyone to view the live site.

## Technologies Used
- **HTML5**: For structuring the content of the website.
- **CSS3**: For styling and layout, enhancing visual appeal.
- **GitHub Pages**: For hosting the website, making it publicly accessible.
- **XAMPP**: For local testing and development environment setup.
